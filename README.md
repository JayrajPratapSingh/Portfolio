# 🚀 Jayraj Portfolio

A futuristic developer portfolio built with modern web technologies, immersive animations and interactive 3D experiences.

Live project focused on creating a premium cyber-style experience with motion, ThreeJS and dynamic content management.

---

## ✨ Features

- 🌌 Interactive ThreeJS scenes
- ⚡ Smooth animations with Framer Motion
- 🎨 Glassmorphism + cyber UI
- 📩 Working contact form with email integration
- 🔐 Admin login panel
- 📱 Fully responsive layout
- 🚀 Modern Navbar + animated sections
- 🎭 React Hook Form integration
- ☄️ Floating effects and immersive visuals

---

```env

```

Add:

```env

JWT_SECRET=

RESEND_API_KEY=

CONTACT_RECEIVER=
```
